### Description

This is a CRUD (Create, Read, Update, Delete) application for managing products, connected to a local REST API.

### Functionality

* Create new products with name, description, and price.
* Display products dynamically in card format.
* Edit existing products through a modal form.
* Delete products from the list.
* Synchronize data with a local API (`json-server`).
* Basic form validation and user feedback.

### Purpose

Practice CRUD operations, asynchronous JavaScript (`fetch`, `async/await`), DOM rendering, and API integration.
